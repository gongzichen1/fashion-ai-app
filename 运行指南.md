## 快速启动

### 第一步：配置智谱AI API密钥

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/) 注册账号
2. 获取API密钥
3. 在 `src/backend/` 目录下复制配置文件：

```bash
cd src/backend
cp .env.example .env
```

4. 编辑 `.env` 文件，填入你的API密钥：

```
ZHIPU_API_KEY=你的API密钥
```

### 第二步：启动后端服务

**macOS/Linux:**
```bash
cd src/backend
./start.sh
```

**Windows:**
```bash
cd src/backend
start.bat
```

**手动启动:**
```bash
cd src/backend
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

服务启动后访问 http://localhost:5000 查看API状态

### 第三步：导入小程序

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. 打开微信开发者工具，选择「导入项目」

3. 选择项目目录：`src/frontend`

4. 在 `project.config.json` 中填入你的小程序AppID（或使用测试号）

5. 修改 `app.js` 中的API地址（如果后端不在本地）：
```javascript
globalData: {
  apiBaseUrl: 'http://你的服务器地址:5000/api'
}
```

6. 点击「编译」运行小程序

## 常见问题

### Q: 后端启动失败？
A: 检查Python版本（需要3.8+）和依赖安装是否完整

### Q: 小程序请求失败？
A:
1. 确保后端服务已启动
2. 在开发者工具中勾选「不校验合法域名」
3. 检查API地址配置是否正确

### Q: AI分析结果不准确？
A:
1. 确保图片清晰、服装主体明显
2. 检查API密钥是否有效
3. 可以在 `ai_service.py` 中调整prompt优化结果
