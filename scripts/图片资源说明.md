## 生成PNG图片

项目提供了SVG格式的图标，但微信小程序TabBar只支持PNG格式。

### 方法一：运行生成脚本

```bash
cd 穿搭ai项目/scripts
pip install Pillow
python generate_images.py
```

### 方法二：在线转换

1. 访问 https://cloudconvert.com/svg-to-png
2. 上传 `src/frontend/images/` 目录下的SVG文件
3. 设置输出尺寸为81x81（TabBar图标）或其他尺寸
4. 下载PNG文件并替换

### 方法三：使用设计工具

1. 使用 Figma / Sketch / Adobe XD 打开SVG文件
2. 导出为PNG格式
3. 按照上述尺寸要求导出
